## Overview

FluxUI is an open-source design system and component library that bridges the gap between design and engineering. It ships with a Figma plugin that auto-syncs design tokens to code.

## What's Included

- **80+ Components** — From primitives to complex data visualization components
- **Figma Plugin** — Two-way sync of design tokens
- **Storybook** — Interactive documentation with live examples
- **Chromatic** — Automated visual regression testing on every PR
- **Accessibility** — WCAG AA compliant across all components
