## Overview

ChainVault is a decentralized treasury management protocol that enables DAOs and Web3 organizations to securely manage funds, execute automated yield strategies, and govern spending via on-chain proposals.

## Features

- **Multi-sig Security** — Gnosis Safe integration with customizable threshold signatures
- **Yield Optimization** — Automated rebalancing across Aave, Compound, and Yearn
- **Cross-chain** — Bridge integration supporting 12 EVM-compatible networks
- **On-chain Governance** — Proposal system with time-locked execution

## Metrics

- **$50M+** in TVL managed at peak
- **2,400+** active DAO treasuries
- **Zero** security incidents since launch
